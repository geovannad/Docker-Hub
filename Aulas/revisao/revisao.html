Revisão

Para relembrar os comandos vistos, agora você precisa baixar uma imagem do ubuntu que está no dockerhub, criar um container a partir dela, e subir a sua imagem para o dockerhub. após isso, você precisa parar o container, removê-lo e logo em seguida remover a imagem.


Clique nos comandos na ordem correta

docker run -- name meuConteiner ubuntu docker stop meuConteiner docker pull ubuntu:latest docker rm meuConteiner docker rmi ubuntu:latest docker push username:minhaImagem
