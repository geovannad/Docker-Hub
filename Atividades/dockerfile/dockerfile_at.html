Atividade - Docker File

1ª questão

Qual é a função do Dockerfile?

Instalar o Docker no sistema operacional. Listar todos os containeres em execução no host. Criar uma nova imagem a partir de um container em execução. Definir os comandos a serem executados dentro de uma imagem.

2ª questão

O que é um Dockerfile?

Um arquivo de registro para eventos de containeres. Um arquivo de script para gerenciar múltiplos containeres. Um arquivo de texto que contém instruções para construir uma imagem Docker. Um arquivo de configuração para o Docker Compose.

3ª questão

Qual é a instrução no Dockerfile para copiar arquivos do host para a imagem Docker?

COPY ADDS MOVE TRANSFER

4ª questão

Qual comando no Dockerfile é usado para definir o diretório de trabalho de um container?

RUN CMD WORKDIR SETDIR

5ª questão

O que a instrução EXPOSE faz no Dockerfile?

Inicia um serviço dentro do container. Especifica qual porta o container estará escutando. Define variáveis de ambiente no container. Executa um comando no container.

6ª questão

Qual é a diferença entre as instruções RUN e CMD no Dockerfile?

Ambas as instruções são usadas para definir o comando padrão a ser executado quando o container for iniciado. Ambas as instruções são usadas para executar comandos durante a construção da imagem. RUN define o comando padrão a ser executado quando o container for iniciado, enquanto CMD executa comandos durante a construção da imagem. RUN executa comandos durante a construção da imagem, enquanto CMD define o comando padrão a ser executado quando o container for iniciado.
