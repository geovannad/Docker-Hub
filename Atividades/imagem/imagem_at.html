Atividade - Imagem

1ª questão

Qual comando Docker é usado para baixar uma imagem do Docker Hub?

docker fetch docker download docker pull docker get

2ª questão

Qual comando Docker é usado para construir uma imagem a partir de um Dockerfile?

docker build docker create docker make docker construct

3ª questão

Como você especifica a versão de uma imagem ao baixá-la?

docker pull ubuntu:version=3.0 docker pull ubuntu ver=3.0 docker pull ubuntu:3.0 docker pull ubuntu-3.0

4ª questão

Qual comando é usado para listar todas as imagens Docker no sistema?

docker list images docker images docker list docker image ls

5ª questão

Como você remove uma imagem Docker do sistema?

docker delete image <image_id> docker remove image <image_id> docker rmi <image_id> docker rm <image_id>

6ª questão

Qual comando é utilizado para renomear uma imagem Docker?

docker rename <old_image><new_image> docker tag <old_image><new_image> docker retag <old_image><new_image> docker rename image <old_image><new_image>
