Atividade - container

1ª questão

Qual comando você usaria para acessar o terminal de um container em execução chamado "ub"?

docker connect -it ub bash docker attach -it ub bash docker exec -it ub bash docker open -it ub bash

2ª questão

Qual comando é usado para mapear a porta 80 do host para a porta 8080 de um container?

docker run -p 8080:80 app docker run -p 80:8080 app docker run --port 8080:80 app docker run --port 80:8080 app

3ª questão

Qual comando exibe todos os containeres, incluindo os que estão parados?

docker ps -all docker ps -a docker ps -p docker ps -list

4ª questão

Qual comando remove um container chamado "banco"?

docker delete banco docker remove banco docker rm banco docker del banco

5ª questão

Qual comando Docker é usado para listar containers em execução?

docker list  docker ps docker containers docker running

6ª questão

Qual comando é usado para rodar um container Docker?

docker start docker run docker init docker launch

7ª questão

Qual comando é usado para parar um container Docker?

docker halt docker stop docker end docker terminate

8ª questão

Qual comando é usado para remover um container Docker?

docker rm docker remove docker delete docker erase

9ª questão

Quando criamos um container com o comando docker run, qual opção podemos usar para especificar um nome para o container?

--name -n --container-name -c

10ª questão

Como podemos especificar a porta do host a ser mapeada para a porta do container ao criar um container?

--port -map-port -p --expose-port

11ª questão

Qual opção do comando docker run é usada para remover automaticamente o container quando ele é encerrado?

--remove --delete --rm -e
