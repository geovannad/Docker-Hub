Atividade

1ª questão

Qual é a função do Dockerfile?

Instalar o Docker no sistema operacional. Listar todos os contêineres em execução no host. Criar uma nova imagem a partir de um contêiner em execução. Definir os comandos a serem executados dentro de uma imagem.

2ª questão

O que é um Dockerfile?

Um arquivo de registro para eventos de contêineres. Um arquivo de script para gerenciar múltiplos contêineres. Um arquivo de texto que contém instruções para construir uma imagem Docker. Um arquivo de configuração para o Docker Compose.

3ª questão

Qual é a instrução no Dockerfile para copiar arquivos do host para a imagem Docker?

COPY ADD MOVE TRANSFER

4ª questão

Qual comando no Dockerfile é usado para definir o diretório de trabalho de um contêiner?

RUN CMD WORKDIR SETDIR

5ª questão

O que a instrução EXPOSE faz no Dockerfile?

Inicia um serviço dentro do contêiner. Especifica qual porta o contêiner estará escutando. Define variáveis de ambiente no contêiner. Executa um comando no contêiner.

6ª questão

Qual é a diferença entre as instruções RUN e CMD no Dockerfile?

Ambas as instruções são usadas para definir o comando padrão a ser executado quando o contêiner for iniciado. Ambas as instruções são usadas para executar comandos durante a construção da imagem. RUN define o comando padrão a ser executado quando o contêiner for iniciado, enquanto CMD executa comandos durante a construção da imagem.
